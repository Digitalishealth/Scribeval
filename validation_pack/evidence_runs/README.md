# Validation Evidence Runs

This directory contains publishable validation evidence-run bundles generated by
`scripts/build_validation_evidence_bundle.py`.

The committed `synthetic_bootstrap_v1/` bundle is illustrative bootstrap data.
It exercises the complete evidence-run shape, source hashing, review-material
provenance, reliability reporting, consensus reporting, stratified agreement
summary, adjudication-burden summary, validation-claim readiness assessment,
and run indexing workflow. It is not independent clinical validation.

Raw reviewer worksheets, reviewer registries, and reviewer-specific assignment
worksheets are intentionally not committed here. Regenerate the synthetic
bundle and index with:

```bash
python scripts/build_synthetic_evidence_bundle.py
python scripts/audit_validation_evidence_runs.py
```
